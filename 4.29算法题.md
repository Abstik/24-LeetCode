# [**62. 不同路径**](https://leetcode.cn/problems/unique-paths/)

![](https://kevin-java.oss-cn-hongkong.aliyuncs.com/2025/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202025-04-29%20215458.png)

```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[] dp = new int[n];
        Arrays.fill(dp, 1);
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[j] = dp[j] + dp[j - 1];
            }
        }
        return dp[n - 1];
    }
}
```
